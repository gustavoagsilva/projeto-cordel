# 📜 Projeto Cordel

## 📌 Descrição do Projeto

O **Projeto Cordel** é uma página web desenvolvida para apresentar conteúdos relacionados à literatura de cordel, incluindo textos, imagens e elementos visuais típicos desse gênero literário.  

O site possui um layout organizado e estilizado, com foco na leitura e na valorização da cultura popular brasileira. A interface é responsiva, permitindo que o conteúdo seja acessado facilmente em diferentes dispositivos, como desktops, tablets e smartphones.

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é praticar e consolidar conhecimentos em **desenvolvimento front-end**, aplicando conceitos de estruturação de páginas, organização de estilos e responsividade.  

Durante o desenvolvimento foram explorados aspectos como:

- Estrutura semântica em HTML
- Criação de layouts organizados e responsivos
- Estilização de conteúdo textual e visual
- Organização de arquivos CSS utilizando boas práticas
- Estruturação de projetos web

---

## 🚀 Funcionalidades

O site apresenta as seguintes funcionalidades:

- Exibição de textos e ilustrações de literatura de cordel
- Layout responsivo adaptável a diferentes dispositivos
- Organização visual clara com seções temáticas
- Conteúdo estruturado para fácil leitura
- Interface simples e intuitiva

---

## 🛠 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** → Estrutura e semântica da página
- **CSS3** → Estilização do conteúdo e layout
- **Flexbox / Grid** → Organização dos elementos na página
- **Git e GitHub** → Controle de versão do projeto
